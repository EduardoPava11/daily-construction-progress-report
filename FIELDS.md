# Field inventory: where you write vs. where you tick

Every input on the form, grouped so you can see at a glance what is **typed**
(writing goes here) versus what is a **checkbox** (you just tick it).

## ✍️ Written / typed fields (112 total)

|---|---|

| Contract No. | text |
| Network Activity No. | text |
| Location (Stage) | text |

### 4. Contractors Workforce grid (85)
A 5-column grid (Prime + 4 Sub Contractors). Each column has 8 rows, and every
row has a **Crew** box and an **Equipment** box.
| Group | Count |
|---|---|
| Contractor name headers (Prime + 4 Sub) | 5 text |
| Crew cells (5 columns × 8 rows) | 40 text |
| Equipment cells (5 columns × 8 rows) | 40 text |

### 5. Work In Progress: descriptions (8)
8 typed lines describing the day's work. Legend above the box:
*1) Excavation 2) Drainage 3) Subbase/Subgrade 4) Concrete 5) Asphalt 6) Utilities 7) Other.*

### 6. Work Outside Contract / Notes (1)
One large multi-line box: *work outside contract, noteworthy conversations,
damage to property, etc.*

|---|---|
| Inspector Signature (typed name) | text |

| Project Manager: Date | date |
